Module: 1

Accepts a POST request from the TTN HTTP integrations uplink API, version 1 as specified here: https://www.thethingsnetwork.org/docs/applications/http/ and decodes it's payload to store a set of measurements for a box. The box is identified by it's registered values app_id and dev_id.
